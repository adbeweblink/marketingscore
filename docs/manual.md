# MarketingScore 活動評分系統 — 操作手冊

---

## 系統簡介

MarketingScore 是一個活動即時互動評分平台。參與者用手機掃 QR Code 加入活動，即時投票評分，大螢幕同步顯示排行榜動畫。

適用場景：尾牙春酒、Team Building、歌唱比賽、才藝表演。

## 一、參與者操作（每位賓客的手機）

### Step 1：掃 QR Code 或開連結

打開連結後看到活動名稱和麥克風圖示。

### Step 2：輸入暱稱

輸入你的名字或暱稱（例如 Mark），按「下一步」。

### Step 3：選桌號

看桌上的號碼牌，點對應的數字，按「下一步」。

### Step 4：確認加入

確認暱稱和桌號無誤，按「確認加入」。

### Step 5：等待

看到等待畫面，表示已成功加入。畫面會自動更新，不需要重新整理。

---

### 投票方式

#### 猜謎制（蒙面歌手）

1. 畫面出現所有桌號選項
2. 你的桌會灰掉不能選
3. 點選你覺得的答案
4. 選了就送出（不能更改）

#### 評分制（團體自選曲、合唱 PK）

1. 畫面出現所有桌號格子
2. 點一個桌號 → 底部彈出評分面板
3. 選 1-10 分 → 點「送出」
4. 逐桌評完，已完成的桌顯示打勾
5. 全部評完自動切到「投票成功」

#### 歡呼制（飆高音挑戰）

1. 手機顯示「用力歡呼」
2. 不需要操作手機
3. 現場大聲歡呼就好
4. 主持人會根據歡呼聲手動加分

---

### 如果畫面卡住

- 點底部的「畫面沒動？點這裡同步」
- 或手動重新整理頁面（不用重新輸入，會自動恢復）

## 二、主持人操作

### 控制台介面說明

- **頂部**：活動名稱 + 已加入人數 + 分享連結按鈕
- **中間**：目前回合狀態 + 控制按鈕
- **下方**：所有回合列表（每個有「開始」按鈕）
- **底部工具列**：大螢幕 / 最終排名 / 複製連結

### 每一輪操作流程

```
1. 點「開始」     → 大家手機自動跳到投票頁
2. 等大家投完     → 看進度條（全投完會變綠色）
3. 長按「結束投票」→ 按住不放 2 秒，進度條跑滿
4. 點「揭曉結果」 → 大螢幕顯示排行榜
5. 點「下一輪」   → 自動進入下一回合
```

---

### 詳細步驟

#### 第 1 步：開始回合

在回合列表找到要開始的回合，點旁邊的「開始」按鈕。大家的手機會在 1-2 秒內自動跳到投票頁。

#### 第 2 步：等待投票

進度條顯示「已投 X / Y 人」。全員投完時進度條變綠色。如果一直有人沒投，可以直接長按結束。

#### 第 3 步：結束投票

長按「結束投票」按鈕不放，看到進度條跑滿（約 2 秒）後放手。

#### 第 4 步：揭曉結果

點金色大按鈕「揭曉結果」。大螢幕顯示排行榜動畫，手機也會顯示精簡排行榜。

#### 第 5 步：下一輪

點「下一輪」，自動進入下一回合。大家手機會自動切換。

---

### 飆高音挑戰（歡呼裁決制）特殊操作

1. 點「開始」
2. 控制台出現每桌的減號和加號按鈕
3. 聽現場歡呼聲，按加號為表現好的桌加分
4. 長按結束 → 揭曉結果

### 最終排名

所有回合結束後，點底部工具列的「最終排名」。大螢幕會顯示金碧輝煌的最終排行榜。

## 三、大螢幕說明

### 畫面狀態

| 狀態 | 顯示內容 |
|------|---------|
| 待機 | 活動名稱 + 大 QR Code + 活動代碼 |
| 投票中 | 回合標題 + 投票進度條 |
| 計分中 | 緊張的分數跳動動畫 |
| 揭曉 | 排行榜從最後一名逐一亮起 |
| 最終排名 | 金碧輝煌 + 完整排行 |

### 注意事項

- 使用 Chrome 瀏覽器
- 按 F11 進入全螢幕
- 解析度建議 1920x1080
- 如果畫面停住，請重新整理頁面

## 五、常見問題

**Q：手機沒自動切換怎麼辦？**
A：點「畫面沒動？點這裡同步」，或手動重新整理。

**Q：有人遲到怎麼加入？**
A：控制台有「分享加入連結」按鈕，可分享 QR Code。

**Q：按錯結束投票怎麼辦？**
A：「結束投票」需要長按 2 秒才觸發，防止誤觸。

**Q：不能投自己桌嗎？**
A：不能，系統自動擋住，自己桌的選項會灰掉。

**Q：可以重複投票嗎？**
A：不行，每人每輪只能投一次。

**Q：關掉瀏覽器再開，要重新輸入嗎？**
A：不用，系統會自動恢復。

**Q：網路不好怎麼辦？**
A：系統每秒自動同步，短暫斷線會自動恢復。建議準備 4G 分享器作為備案。

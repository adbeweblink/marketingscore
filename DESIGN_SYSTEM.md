## 1. 色彩系統

| 名稱 | Hex | 用途 |

| **Gold Primary** | `#FFD700` | 主標題、冠軍高亮、重要數字 |
| **Gold Light** | `#FFE55C` | hover 狀態、活躍指標、進度條填充 |
| **Gold Deep** | `#C9A019` | 次要標題、邊框、icon 填色 |
| **Amber Glow** | `#F5A623` | 第二名標記、警告色、CTA 按鈕 |
| **Bronze** | `#CD7F32` | 第三名標記、輔助裝飾 |

### 背景色盤（Dark Stage）

| 名稱 | Hex | 用途 |
|------|-----|------|
| **Background Primary** | `#0A0A14` | 大螢幕主背景（近乎純黑帶微紫） |
| **Background Elevated** | `#12121F` | 卡片背景、控制台面板 |

| **Border Subtle** | `#2A2A3E` | 分隔線、低對比邊框 |

### 漸層

```css
/* 金色漸層 — 標題背景、冠軍卡片 */
--gradient-gold: linear-gradient(135deg, #FFD700 0%, #F5A623 50%, #C9A019 100%);

/* 深色漸層 — 大螢幕背景 */
--gradient-stage: radial-gradient(ellipse at center bottom, #1A1A2E 0%, #0A0A14 70%);

/* 光暈漸層 — 聚光燈效果 */
--gradient-spotlight: radial-gradient(circle at 50% 30%, rgba(255,215,0,0.15) 0%, transparent 60%);

/* 排名色帶 */
--rank-1: linear-gradient(135deg, #FFD700, #FFF176);  /* 金 */
--rank-2: linear-gradient(135deg, #B0BEC5, #E0E0E0);  /* 銀 */
--rank-3: linear-gradient(135deg, #CD7F32, #FFAB40);  /* 銅 */
```

### 大螢幕 vs 手機差異

| 屬性 | 大螢幕（投影） | 手機（LIFF） |
|------|--------------|-------------|
| 背景 | `#0A0A14` 純深色（對比投影環境） | `#12121F` 微亮（省電+可讀） |
| 主色 | Gold Primary `#FFD700`（最高亮度） | Gold Deep `#C9A019`（防刺眼） |
| 文字 | 純白 `#FFFFFF`（遠距離可讀） | `#E8E8F0`（近距離柔和） |
| 裝飾 | 大量粒子+光效+漸層 | 極簡，僅 icon + 色塊 |

### 投影可讀性設計原則

1. **最小對比度 7:1**（WCAG AAA），分數/桌號對比度 > 12:1
2. **文字不依賴漸層**：漸層只用在裝飾元素，關鍵文字一律純色
3. **安全邊距**：畫面四周留 5% padding，避免投影機裁切
4. **無純藍/純紅小字**：投影機色偏嚴重，小字只用白/金/淺灰

## 2. 字體方案

### 大螢幕字體堆疊

```css
/* 數字/分數 — 粗體等寬，數字跳動時不位移 */
--font-score: 'Inter', 'Noto Sans TC', sans-serif;
font-variant-numeric: tabular-nums;

/* 標題 */
--font-display: 'Inter', 'Noto Sans TC', sans-serif;

/* 內文 */
--font-body: 'Noto Sans TC', 'Microsoft JhengHei', sans-serif;
```

### 手機字體堆疊

```css
--font-mobile: 'Noto Sans TC', -apple-system, 'Microsoft JhengHei', sans-serif;
```

### 字體大小（大螢幕 1920x1080）

| 層級 | 大小 | 用途 | 距離可讀性 |
|------|------|------|-----------|
| **Display XL** | 120px / 7.5rem | 冠軍分數、最終結果 | 30m+ |
| **Display L** | 96px / 6rem | 回合分數、倒數計時 | 25m+ |
| **Display M** | 72px / 4.5rem | 桌號、排名數字 | 20m+ |
| **Heading 1** | 48px / 3rem | 回合標題、頁面標題 | 15m+ |
| **Heading 2** | 36px / 2.25rem | 副標題、說明文 | 12m+ |
| **Body L** | 24px / 1.5rem | 狀態訊息、已投票數 | 8m+ |
| **Body M** | 18px / 1.125rem | 最小可讀文字 | 5m+ |

### 字體大小（手機 375px 寬）

| 層級 | 大小 | 用途 |
|------|------|------|
| **Title** | 28px / 1.75rem | 頁面標題 |
| **Score** | 48px / 3rem | 分數顯示 |
| **Body** | 16px / 1rem | 一般內文 |
| **Caption** | 14px / 0.875rem | 說明、提示 |
| **Small** | 12px / 0.75rem | 法律文字、版本號 |

### 中文字體推薦（免費商用）

| 字體 | 授權 | 特色 | 用途 |
|------|------|------|------|
| **Noto Sans TC** | OFL（Google 免費） | 最完整繁中支援、多字重 | 主力字體 |
| **Noto Serif TC** | OFL | 宋體風格 | 冠軍頁裝飾標題（可選） |
| **Inter** | OFL | 等寬數字、極佳可讀性 | 分數、計數器、數據 |

> Inter 的 `tabular-nums` 特性確保分數跳動時數字不左右位移，這對計分動畫至關重要。

- 接近尾聲時故意放慢（製造緊張感）
- 前 3 名咬很近時加入心跳音效
- 最終分數出來瞬間：第一名金光爆閃 + 粒子噴射

## 4. 動畫設計規格

### 4.1 分數累計動畫（緊張感核心）

```
Timeline（約 10 秒）:
════════════════════════════════════════════════════

0s────2s────4s────6s────8s────10s
 │  Phase 1  │  Phase 2  │  Phase 3  │ Phase 4│
 │  起跑      │  加速      │  緊咬      │ 定格   │

Phase 1 — 起跑（0-2.5s）
  easing: cubic-bezier(0.25, 0.1, 0.25, 1.0)  // ease-out
  所有分數條同時從 0 開始跑
  速度：慢（目標值的 30%）
  音效：低沉鼓聲滾動（drum roll）
  視覺：進度條平穩增長

Phase 2 — 加速（2.5-5s）
  easing: cubic-bezier(0.4, 0.0, 0.2, 1)  // ease-in-out
  速度加快到目標值 70%
  差距開始拉開
  音效：鼓聲加快
  視覺：領先的條開始明顯突出

Phase 3 — 緊咬（5-8s）
  easing: cubic-bezier(0.0, 0.0, 0.2, 1)  // ease-out slow
  故意放慢！接近最終值但還在微調
  前 3 名之間差距很小（視覺上看起來在追趕）
  音效：心跳聲（thump-thump），BPM 逐漸加快
  視覺：前 3 名的條閃爍金色邊框

Phase 4 — 定格（8-10s）
  easing: step-end
  所有分數停在最終值
  第一名的條金色爆閃 + 放大 1.05x
  音效：號角 fanfare（ta-da!）
  視覺：第一名金色粒子噴射，其餘淡出到 80% opacity
```

**Framer Motion 實作概念**：
```jsx
// 分數數字跳動
<motion.span
  animate={{ '--score': finalScore }}
  transition={{
    duration: 10,
    ease: [0.25, 0.1, 0.25, 1.0],  // 自訂曲線
    times: [0, 0.25, 0.5, 0.8, 1], // keyframe 時間點
  }}
/>

// 進度條
<motion.div
  animate={{ width: `${(finalScore / maxScore) * 100}%` }}
  transition={{ duration: 10, ease: customEasing }}
/>
```

### 4.2 排名揭曉動畫

```
每一名的揭曉序列（1.5s / 名，前 3 名 3s / 名）:

0ms     ── 空位出現（灰色佔位條）
200ms   ── 排名數字從左滑入
400ms   ── 桌號從模糊→清晰（blur 0→clear）
600ms   ── 分數從右飛入
800ms   ── 整行亮起（opacity 0.5→1）
1000ms  ── 下一名開始

前 3 名額外效果：
  #3: 銅色光暈 (box-shadow: 0 0 30px #CD7F32)
  #2: 銀色光暈 (box-shadow: 0 0 40px #B0BEC5)
  #1: 金色光暈 (box-shadow: 0 0 60px #FFD700) + 放大 1.1x + 粒子爆發
```

### 4.3 投票即時回饋

```
手機端（投票者）:
  送出按鈕 → 按下縮小 0.95x → 放開彈回 1.0x → 成功 ✓ 打勾動畫（0.3s）
  失敗 → 按鈕搖晃 (translateX 左右 5px × 3次) → 紅色閃一下

大螢幕（觀眾）:
  每收到一票 → 對應桌的進度 +1
                → 飄出 "+1" 數字（0.5s 向上飄 + fade out）
                → 進度條微增長（0.2s ease-out）
```

### 4.4 轉場效果

```
頁面切換（大螢幕）:
  淡出當前頁（0.3s, opacity 1→0）
  → 金色光幕從左掃到右（0.4s）
  → 淡入新頁（0.3s, opacity 0→1）
  總時長：1.0s

回合切換特效：
  「第 N 輪」大字從中央放大出現（scale 0→1, 0.5s）
  → 停留 1.5s → 縮小消失
  → 進入回合詳情頁
```

## 5. 匿名/具名模式的 UI 差異

### 匿名模式（預設，評分/投票時）

**手機端**：
- 投票後顯示「你的評分已送出 ✓」，不顯示其他人的票
- 等待頁不顯示「誰投了什麼」

**大螢幕**：
- 進度條只顯示「已投 68/96 人」，不顯示個人投票內容
- 分數揭曉時只顯示桌的平均分，不顯示是誰投的
- 各桌完成度只顯示 ✓/○，不列具體人名

```
匿名模式顯示：
┌─────────────────────────┐
│ 第 1 桌  ████████  8.3  │ ← 只有桌+分數
│ 第 5 桌  ███████   7.9  │
│ 第 3 桌  ██████    7.2  │
└─────────────────────────┘
```

### 具名模式（主持人裁決制、特殊場景）

**手機端**：
- 投票卡片顯示「第 X 桌 — 桌名」
- 可選：顯示桌員 LINE 頭像（需 SPEC 確認隱私政策）

**大螢幕**：
- 排行榜顯示桌號 + 桌名（如果有設定）
- 冠軍頁可顯示桌員名字（設定決定）

```
具名模式顯示：
┌──────────────────────────────────┐
│ 🥇 第 3 桌 "Photoshop 隊"       │
│    67.8 分                       │
│    👤 Amy 👤 Bob 👤 Chris ...    │ ← 顯示成員（可選）
└──────────────────────────────────┘
```

### 切換機制

- 在「活動設定」中全域設定：匿名/具名
- 每個回合可獨立覆蓋（某輪匿名、某輪具名）
- 預設：投票階段匿名、結果階段顯示桌號（不顯示個人）
